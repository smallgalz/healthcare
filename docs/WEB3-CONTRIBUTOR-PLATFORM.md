# 🌟 MediChain - Web3 Contributor Platform

## 🎯 Platform Overview

MediChain is a **Web3 decentralized platform** that enables contributors worldwide to participate in healthcare insurance funding through **Stellar blockchain technology**. Contributors can review, approve, and fund medical treatment requests while earning rewards.

## 🎯 Contribution Process

### **📝 Application Workflow**

#### **Step 1: Browse Issues**
```typescript
// View available issues
const issues = await contract.get_active_issues();
const filteredIssues = issues.filter(issue => 
  issue.issue_type === contributor.specialty
);
```

#### **Step 2: Apply to Review**
```typescript
// Submit application
await contract.apply_to_issue(
  issue_id,
  "I have 10 years of experience in cardiac surgery and can provide expert review",
  contributor_address
);
```

#### **Step 3: Review Process**
```typescript
// Submit review
await contract.review_application(
  issue_id,
  contributor_address,
  true, // approved
  "Medical documentation is complete and surgery is medically necessary",
  reviewer_address
);
```

#### **Step 4: Earn Rewards**
```typescript
// Rewards automatically distributed
const reputation = await contract.get_contributor_stats(contributor_address);
const rewards = reputation.total_issues_approved * get_level_rewards(reputation.level);
```
